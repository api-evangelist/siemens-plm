# Siemens PLM

Siemens PLM is Siemens Digital Industries Software's product lifecycle management platform offering REST APIs for Teamcenter (PDM/PLM), Polarion (ALM), Insights Hub (Industrial IoT), SiGREEN (sustainability), and the broader Siemens Xcelerator portfolio for digital manufacturing, simulation via Simcenter, and CAD/CAM tooling including NX and Solid Edge.

**Developer Portal:** https://developer.siemens.com/

## APIs

| API | Description |
|-----|-------------|
| [Teamcenter Product Cost Management REST API](https://developer.siemens.com/tcpcm/overview.html) | Cost estimation and bulk export for Teamcenter X PCM |
| [Polarion REST API](https://developer.siemens.com/polarion/rest-api-spec.html) | ALM integration for requirements, work items, and test management |
| [Insights Hub Industrial IoT APIs](https://developer.siemens.com/apis.html) | Time series, asset management, and Opcenter Intelligence |
| [SiGREEN API](https://developer.siemens.com/sigreen/product-api/api-product.html) | Supply chain carbon footprint and sustainability management |
| [Product Information API](https://developer.siemens.com/product-information-api/rest-api-spec.html) | CMMS product information for obsolescence risk mitigation |
